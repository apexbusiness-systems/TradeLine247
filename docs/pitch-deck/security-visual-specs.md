# Security Slide Visual Specifications

## Layout
- **Format:** 16:9 widescreen
- **Background:** Dark gradient (navy blue to black)
- **Theme:** Professional, technical, trustworthy

## Left Column - Metrics (40% width)
### Security Scorecard
- Large "97/100" number with progress circle
- Gold/yellow accent color for grade
- Subtext: "Security Grade" in white

### Key Badges (stacked vertically)
1. **100% RLS Coverage** - Green checkmark badge
2. **A+ SSL Rating** - Blue shield badge
3. **0 Critical CVEs** - Red (crossed out) badge

## Right Column - Details (60% width)

### Infrastructure Hardening (Top Section)
- Icon: Server rack or database icon
- Bullet points with icons
- Font: Clean sans-serif (not Inter/Roboto)
- Color: Light gray text on dark background

### Data Protection (Middle Section)
- Icon: Lock or shield
- Green checkmarks for each item
- Subtle glow effect on checkmarks

### Uptime & Reliability (Bottom Section)
- Icon: Uptime graph or heartbeat
- Blue accent color
- Metrics in large, bold numbers

## Typography
- **Heading:** 48pt, bold, white
- **Subheadings:** 32pt, semibold, light gray
- **Body:** 18pt, regular, gray
- **Metrics:** 64pt, bold, accent color

## Color Palette
- **Primary:** #0A1929 (dark navy)
- **Accent:** #FFB800 (gold/yellow)
- **Success:** #10B981 (green)
- **Info:** #3B82F6 (blue)
- **Text:** #E5E7EB (light gray)
- **White:** #FFFFFF

## Assets Needed
- Security badge icon (97/100)
- Shield icon
- Lock icon
- Server icon
- Uptime graph illustration
- Checkmark icons (vector)

## Tools
- Canva (recommended for quick creation)
- Figma (for more control)
- Export as PNG (high resolution, 1920x1080px)
